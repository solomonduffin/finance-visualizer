  - `flex items-center gap-2 pt-4 pb-2`

  - Label: `text-sm font-semibold uppercase tracking-wide` with panel color

- Standard account row (no holdings OR non-investment account):
  - Grid: `grid grid-cols-[1fr_80px_80px_64px] gap-2 items-center py-2 border-b border-gray-100 dark:border-gray-700/50`
  - Account name: `text-sm text-gray-800 dark:text-gray-200 truncate`
  - APY input: `<input type="text" inputMode="decimal" placeholder="0.0">` with classes `w-full px-2 py-1.5 border border-gray-300 dark:border-gray-600 rounded-md text-sm text-right text-gray-900 dark:text-gray-100 bg-white dark:bg-gray-800 placeholder-gray-400 dark:placeholder-gray-500 focus:outline-none focus:ring-2 focus:ring-blue-500 focus:border-transparent` and `aria-label={`APY for ${account_name}`}`
  - "%" suffix: `text-xs text-gray-500 dark:text-gray-400` placed after input in a flex wrapper
  - Compound toggle: miniature toggle switch `w-8 h-[18px]` with `w-[14px] h-[14px]` knob. On (compound): `bg-blue-600 dark:bg-blue-500`. Off (simple): `bg-gray-300 dark:bg-gray-600`. `role="switch"`, `aria-checked={compound}`, `aria-label="Compound interest for {account_name}"`, `title={compound ? 'Compound' : 'Simple'}`
  - Include checkbox: `<input type="checkbox" checked={included}>` with classes `w-4 h-4 rounded border border-gray-300 dark:border-gray-600 text-blue-600 focus:ring-2 focus:ring-blue-500 accent-blue-600` and `aria-label="Include {account_name} in projection"`
- Investment account row WITH holdings (holdings.length > 0):
  - Same grid as standard row
  - Account name area: `flex items-center gap-1 cursor-pointer` with chevron SVG (16x16, matching GroupRow) that rotates 90deg on expand
  - Balance shown after name: `text-xs text-gray-500 dark:text-gray-400 ml-2` with formatCurrency
  - APY input: disabled/hidden (rates are per-holding)
  - Compound toggle: hidden (per-holding)
  - Include checkbox: master toggle -- unchecking cascades to all holdings via onIncludeChange then onHoldingIncludeChange for each holding
  - `role="button"`, `tabIndex={0}`, `aria-expanded`, `aria-controls={`holdings-${account_id}`}`, Enter/Space toggle
  - Renders `<HoldingsRow>` below when expanded
- Investment account WITHOUT holdings: rendered as standard row (no chevron, account-level rate)
- Expansion state: managed locally with `expandedAccounts: Set<string>` state
- Loading state: 6 skeleton rows of `h-10 bg-gray-200 dark:bg-gray-700 rounded animate-pulse`
- Empty state: "No accounts found. Sync your accounts first." in muted text
- Mobile (<640px): switch to 2-row card layout per UI-SPEC responsive spec:
  - Row 1: `grid grid-cols-[1fr_auto]` with account name + include checkbox
  - Row 2: APY input + compound toggle inline
