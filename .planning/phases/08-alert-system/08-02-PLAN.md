Import `github.com/solomon/finance-visualizer/internal/alerts`

Request/response types:
```go
type createAlertRequest struct {
    Name             string          `json:"name"`
    Operands         json.RawMessage `json:"operands"`
    Comparison       string          `json:"comparison"`
    Threshold        string          `json:"threshold"`
    NotifyOnRecovery bool            `json:"notify_on_recovery"`
}

type alertRuleResponse struct {
    ID               int                  `json:"id"`
    Name             string               `json:"name"`
    Operands         json.RawMessage      `json:"operands"`
    Expression       string               `json:"expression"`
    Comparison       string               `json:"comparison"`
    Threshold        string               `json:"threshold"`
    NotifyOnRecovery bool                 `json:"notify_on_recovery"`
    Enabled          bool                 `json:"enabled"`
    LastState        string               `json:"last_state"`
    LastEvalAt       *string              `json:"last_eval_at"`
    LastValue        *string              `json:"last_value"`
    CreatedAt        string               `json:"created_at"`
    UpdatedAt        string               `json:"updated_at"`
    History          []alertHistoryEntry   `json:"history"`
}

type alertHistoryEntry struct {
    ID         int    `json:"id"`
    State      string `json:"state"`
    Value      *string `json:"value"`
    NotifiedAt string `json:"notified_at"`
}
```

Handlers (all take `database *sql.DB` and return `http.HandlerFunc`):
- `CreateAlert(database)`: decode createAlertRequest, validate name not empty, call alerts.CompileOperands(req.Operands, req.Comparison, req.Threshold), INSERT into alert_rules, return 201 with alertRuleResponse
- `ListAlerts(database)`: SELECT all alert_rules ORDER BY created_at DESC, for each rule SELECT alert_history WHERE rule_id=? ORDER BY notified_at DESC LIMIT 10, return JSON array (empty array [] not null when no rules)
- `UpdateAlert(database)`: decode createAlertRequest, get ID from chi.URLParam(r, "id"), validate, recompile operands, UPDATE alert_rules SET name=?, operands=?, expression=?, comparison=?, threshold=?, notify_on_recovery=?, updated_at=CURRENT_TIMESTAMP WHERE id=?, return updated rule
- `ToggleAlert(database)`: decode `{"enabled": bool}`, PATCH alert_rules SET enabled=?, updated_at=CURRENT_TIMESTAMP WHERE id=?, return updated rule
- `DeleteAlert(database)`: get ID from chi.URLParam, DELETE FROM alert_rules WHERE id=?, return 204

Helper: `boolToInt(b bool) int` returns 1 if true, 0 if false. `intToBool(i int) bool` returns i != 0.
