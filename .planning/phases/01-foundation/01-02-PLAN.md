    - TestProtectedRoute_NoAuth: GET /api/health without cookie, assert 401
    - TestProtectedRoute_WithAuth: GET /api/health with valid JWT cookie named "jwt", assert 200
    - TestProtectedRoute_ExpiredToken: GET /api/health with expired JWT, assert 401

  <action>
    1. Create `internal/api/handlers/health.go`:
       - `Health(w, r)` — returns `{"status":"ok"}` with 200
    2. Create `internal/api/handlers/auth.go`:
       - `Login(db *sql.DB) http.HandlerFunc` — closure over db connection
       - Decodes JSON body `{"password": "..."}` — returns 400 on parse failure
       - Reads password hash from settings table: `SELECT value FROM settings WHERE key = 'password_hash'`
       - Calls `auth.VerifyPassword(storedHash, req.Password)` — returns 401 on mismatch
       - Calls `auth.CreateToken()` to generate JWT
       - Sets HttpOnly cookie: Name="jwt", Path="/", HttpOnly=true, SameSite=Strict, MaxAge=30*24*3600
       - Returns `{"ok":true}` with 200
    3. Create `internal/api/router.go`:
       - `NewRouter(tokenAuth *jwtauth.JWTAuth, db *sql.DB) chi.Router`
       - Middleware: httplog.RequestLogger, cors.Handler (AllowedOrigins: localhost:5173 for dev, AllowCredentials: true)
       - Public group: `r.With(httprate.LimitByIP(5, 30*time.Second)).Post("/api/auth/login", handlers.Login(db))`
       - Protected group: `jwtauth.Verifier(tokenAuth)` + `jwtauth.Authenticator(tokenAuth)` middleware
       - Protected routes: `GET /api/health` -> handlers.Health
       - Rate limiting applied ONLY to login route, not globally
    4. Update `cmd/server/main.go`:
       - After DB open + migrate: seed password hash into settings table if not exists
         (INSERT OR IGNORE INTO settings (key, value) VALUES ('password_hash', config.PasswordHash))
       - Initialize auth: `auth.Init(config.JWTSecret)`
       - Create router: `api.NewRouter(auth.TokenAuth(), db)`
       - Start HTTP server: `http.ListenAndServe(":"+config.Port, router)`
       - Log startup message with port number
    5. Write tests FIRST (RED), then implement (GREEN)
    6. Handler tests use httptest.NewRecorder + httptest.NewRequest
    7. Router tests use a real chi router with in-memory SQLite for integration testing
