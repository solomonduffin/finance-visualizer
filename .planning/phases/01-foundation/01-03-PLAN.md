<task type="auto">
  <name>Task 2: Docker environment -- Dockerfile, Compose files, Nginx, and .gitignore</name>
  <files>Dockerfile, docker-compose.yml, docker-compose.prod.yml, nginx/nginx.dev.conf, nginx/nginx.prod.conf, .air.toml, .gitignore, .dockerignore</files>
  <action>
    1. Create `.gitignore`:
       - Go: /data/, *.db, *.db-wal, *.db-shm, tmp/
       - Node: frontend/node_modules/, frontend/dist/
       - IDE: .vscode/, .idea/
       - Env: .env (but NOT .env.example)
       - Docker: not ignored (tracked)
    2. Create `.dockerignore`:
       - .git, frontend/node_modules, data/, tmp/, .env
    3. Create `Dockerfile` (multi-stage):
       - Stage "dev": FROM golang:1.23-alpine, install Air (`go install github.com/air-verse/air@latest`), WORKDIR /app, CMD ["air"]
       - Stage "frontend-build": FROM node:22-alpine, WORKDIR /app, COPY frontend/package*.json, npm ci, COPY frontend/, npm run build
       - Stage "backend-build": FROM golang:1.23-alpine, WORKDIR /app, COPY go.mod go.sum, go mod download, COPY ., CGO_ENABLED=0 go build -o server ./cmd/server
       - Stage "prod": FROM gcr.io/distroless/static-debian12, COPY --from=backend-build /app/server, COPY --from=frontend-build /app/dist /app/dist, ENTRYPOINT ["/server"]
       - Stage "nginx-prod": FROM nginx:1.25-alpine, COPY --from=frontend-build /app/dist /usr/share/nginx/html, COPY nginx/nginx.prod.conf /etc/nginx/conf.d/default.conf
    4. Create `.air.toml`:
       - root = "."
       - cmd = "go build -o ./tmp/server ./cmd/server && ./tmp/server"
       - Watch: include_ext = ["go", "sql"], exclude_dir = ["tmp", "frontend", "node_modules", "data"]
    5. Create `nginx/nginx.dev.conf`:
       - Listen 80
       - Location /api/ -> proxy_pass http://backend:8080 (with Host, X-Real-IP, X-Forwarded-For headers)
       - Location / -> proxy_pass http://frontend:5173 (with WebSocket Upgrade headers for HMR)
       - CRITICAL: Include `proxy_set_header Upgrade $http_upgrade; proxy_set_header Connection "Upgrade";` for both / and any HMR paths
    6. Create `nginx/nginx.prod.conf`:
       - Listen 80
       - Location /api/ -> proxy_pass http://backend:8080
       - Location / -> root /usr/share/nginx/html, try_files $uri $uri/ /index.html (SPA fallback)
    7. Create `docker-compose.yml` (dev):
       - backend: build context ., target dev, command "air", volumes (.:/app), env (PORT, JWT_SECRET, PASSWORD, DB_PATH=/data/finance.db), sqlite_data:/data
       - frontend: image node:22-alpine, working_dir /app, command "npm run dev -- --host 0.0.0.0", volumes (./frontend:/app, /app/node_modules), environment CHOKIDAR_USEPOLLING=true
       - nginx: image nginx:1.25-alpine, volumes (nginx.dev.conf), ports 80:80, depends_on backend + frontend
       - volumes: sqlite_data (named volume for persistence)
    8. Create `docker-compose.prod.yml`:
       - backend: build context ., target prod, env (PORT, JWT_SECRET, PASSWORD_HASH, DB_PATH=/data/finance.db), volumes sqlite_data:/data, restart unless-stopped
       - nginx: build context ., target nginx-prod, ports 80:80, depends_on backend, restart unless-stopped
       - volumes: sqlite_data
    9. Create `.env.example` (if not already created, or update):
       - PASSWORD=changeme
       - JWT_SECRET=change-me-to-random-64-chars
       - PORT=8080
       - DB_PATH=data/finance.db
  </action>
